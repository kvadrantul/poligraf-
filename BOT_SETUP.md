# Пошаговая инструкция по созданию и настройке Telegram бота

## Шаг 1: Создание бота (если еще не создан)

1. Откройте Telegram и найдите [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/start`
3. Отправьте команду `/newbot`
4. Введите имя бота (например: `Poligraf Bot`)
5. Введите username бота (должен заканчиваться на `bot`, например: `poligraf_bot`)
6. **Сохраните токен бота** - он будет выглядеть как `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`
   - ⚠️ **ВАЖНО**: Токен нужен только для backend (если будете делать серверную часть)
   - Для Mini App токен НЕ нужен в коде!

## Шаг 2: Настройка GitHub Pages (если еще не сделано)

1. Перейдите на: https://github.com/kvadrantul/poligraf-/settings/pages
2. В разделе "Source":
   - Branch: `main`
   - Folder: `/ (root)`
3. Нажмите "Save"
4. Подождите 1-2 минуты
5. Ваш URL: `https://kvadrantul.github.io/poligraf-/`

## Шаг 3: Настройка Mini App в боте

1. Откройте [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/mybots`
3. Выберите вашего бота из списка
4. Выберите "Bot Settings"
5. Выберите "Menu Button"
6. Нажмите "Configure Menu Button" или "Edit Menu Button"
7. Заполните:
   - **Text**: `Открыть Poligraf` (или любое другое название)
   - **URL**: `https://kvadrantul.github.io/poligraf-/`
8. Нажмите "Done" или "Save"

## Шаг 4: Тестирование

1. Откройте вашего бота в Telegram
2. Внизу экрана должна появиться кнопка "Открыть Poligraf"
3. Нажмите на кнопку - Mini App откроется!

## Что дальше?

- Mini App уже работает и отображает интерфейс
- Сейчас кнопка "Отправить" только выводит в консоль
- Для реальной отправки комментариев нужно будет добавить backend (серверную часть)
- Токен бота понадобится только для backend, если будете делать серверную часть

## Безопасность

⚠️ **НИКОГДА не добавляйте токен бота в клиентский код (HTML/JS)!**
- Токен должен использоваться только на сервере
- Mini App работает через Telegram Web App API без токена
