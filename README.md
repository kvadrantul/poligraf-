# Poligraf - Telegram Mini App

Telegram мини-приложение с интерфейсом для отображения результата и отправки комментариев.

## Структура проекта

- `index.html` - основной HTML файл приложения
- `styles.css` - стили для интерфейса
- `app.js` - JavaScript логика с интеграцией Telegram Web App API

## Функциональность

- Поле для отображения результата (верхняя часть)
- Текстовое поле для ввода комментария (нижняя часть)
- Кнопка отправки комментария
- Поддержка тем Telegram (светлая/темная)
- Адаптивный дизайн

## Как использовать

1. Разместите файлы на веб-сервере (например, GitHub Pages, Netlify, или любой другой хостинг)
2. Создайте бота в Telegram через [@BotFather](https://t.me/BotFather)
3. Настройте Web App URL в настройках бота
4. Откройте бота в Telegram и запустите Mini App
