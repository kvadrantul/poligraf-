# Быстрый деплой без командной строки

## Вариант 1: Vercel через GitHub (Самый простой!)

### Шаг 1: Закоммитьте и запушьте код

Я помогу вам закоммитить все файлы в репозиторий.

### Шаг 2: Подключите репозиторий к Vercel

1. Откройте [Vercel](https://vercel.com) и войдите через GitHub
2. Нажмите **"Add New Project"** (Добавить новый проект)
3. Выберите репозиторий `kvadrantul/poligraf-`
4. Нажмите **"Import"**

### Шаг 3: Добавьте API ключ v0.dev

**Перед деплоем:**

1. В разделе **"Environment Variables"** (Переменные окружения)
2. Нажмите **"Add"**
3. Добавьте:
   - **Name**: `V0_API_KEY`
   - **Value**: ваш API ключ от v0.dev (пришлите мне, я помогу правильно вставить)
4. Нажмите **"Save"**

### Шаг 4: Деплой

1. Нажмите **"Deploy"**
2. Дождитесь завершения (обычно 1-2 минуты)
3. Скопируйте URL вашего проекта (например: `https://poligraf-xxx.vercel.app`)

### Шаг 5: Обновите app.js

После получения URL от Vercel, я помогу обновить `app.js` с правильным URL.
